## 注意点

* 为防止业主签名表被人伪造签名，需业主提供**产权证号**
* **产权证号**可以是房产证号，也可以是房屋预售证号
* 筹备组成员可以是产权业主，也可以是非产权业主
* 业委会成员必须是产权业主
* 成立业主大会筹备组必须超过 **50%** 业主签名
* 筹备组应积极主动联系业主，对于未入住业主可要求物业当面联系业主咨询是否愿意提供联系方式给筹备组
